Keyboard Shortcuts: For New Programmers

Mac

Highlight Code from the Cursor to the Start or End of the Line:
Select Code Right of the Cursor: ⌘ + Shift + Right Arrow
Select Code Left of the Cursor: ⌘ + Shift + Left Arrow

Cut: ⌘ + X
Copy: ⌘ + C
Paste: ⌘ + V
Save: ⌘ + S
Comment Out Lines of Code: ⌘ + /
Shift Right: Tab
Shift Left: Shift + Tab
Toggle Tabs to the Right: CTRL + Tab
Toggle Tabs to the Left: CTRL + Shift + T
Open a Find Bar: ⌘ + F
Select Matching Instances One at a Time: ⌘ + D
Select All Matching Instances at Once: ⌘ + Shift + L

Highlight an Entire File: ⌘ + A

Screenshot - Entire Screen: ⌘ + Shift + 3
Screenshot - Portion of Screen: ⌘ + Shift + 5
